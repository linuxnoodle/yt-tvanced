# YouTube TV Enhanced (Electron Edition)

A **completely AI-generated** cross-platform YouTube TV client that brings the powerful TizenTube features (Ad Blocking, SponsorBlock, etc.) to desktop PCs. This project was created to address the significant shortcomings of Kodi's YouTube implementation, providing a superior desktop experience for YouTube TV with enhanced privacy and usability features.

## üöÄ The AI-Generated Story

This entire project - from the Electron application structure to the TizenTube feature integration - was **automatically generated by AI**. The goal was to create a desktop YouTube TV experience that:

- **Eliminates ads** completely from the YouTube TV interface
- **Skips sponsor segments** automatically using SponsorBlock
- **Provides better performance** than Kodi's YouTube implementation
- **Offers full configuration** for all features
- **Works cross-platform** on Windows, macOS, and Linux

The AI analyzed existing TizenTube implementations, understood the YouTube TV web interface, and created this Electron-based solution to bring smart TV features to desktop computers.

- **Ad Blocking**: Completely removes all ads from the YouTube TV interface

This project uses **electron-builder** to create platform-specific executables. Here's how to build for different platforms:

```bash

```

### Build for Specific Platforms
```bash

# macOS (DMG)
npm run dist -- --mac

# Linux (AppImage)
npm run dist -- --linux
```

### Build Configuration
The build process creates:
- **Windows**: NSIS installer with custom icon
- **macOS**: DMG package with proper signing support
- **Linux**: AppImage for easy distribution

All builds include:
- Automatic updates via electron-updater
- Proper code signing (configure in package.json)
- Platform-specific icons
- Persistent configuration storage

### Keyboard Shortcuts
- **F11**: Toggle fullscreen
- **Esc**: Exit fullscreen
- **Arrow Keys**: Navigation
- **Enter**: Select
- **Space**: Play/Pause

### Configuration
The app stores all configuration in a persistent store. You can access settings through:

1. **Settings Menu**: Click the gear icon in the interface
2. **Configuration File**: Located in your system's app data directory
3. **IPC Commands**: Advanced users can use Electron's IPC for configuration
